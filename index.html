CSS Selectors Reference


CSS Selectors

Selector Name Description
* Universal Selector Selects all elements of the HTML document.
elementname Type Selector Selects all elements that have the given node name.
.classname Class Selector Selects all elements that have the given class attribute.
#idname ID Selector Selects an element based on the value of its id attribute.
[attribute] Attribute Selector Selects all elements that have the given attribute.
, / A, B Grouping Selector Selects all the matching nodes.

CSS Combinators

Combinator Name Description
A B Descendant Combinator The (space) combinator selects nodes that are descendants of the first element.
> Child Combinator The > combinator selects nodes that are direct children of the first element.
- General Sibling Combinator The - combinator selects siblings. The second element follows the first.
+ Adjacent Sibling Combinator The + combinator matches the second element only if it immediately follows the first element.
|| Column Combinator The || combinator selects nodes which belong to a columm.
